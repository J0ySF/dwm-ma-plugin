# Simple JUCE plugin wrapper around [Digital Waveguide Mesh with Microphone Array output](https://github.com/J0ySF/dwm-ma)

"All in one" testing branch, meant only for future reference.

This version incorporates the backend
of [SPARTA Array2SH](https://leomccormack.github.io/sparta-site/docs/plugins/sparta-suite/#array2sh)
and [SPARTA AmbiBIN](https://leomccormack.github.io/sparta-site/docs/plugins/sparta-suite/#ambibin) inside one single 
plugin.
